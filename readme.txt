== Description ==

Ajax contact form is a simple and clean deisnged contact form. You will be publishing your Ajax Contact Form plugin with just a few steps.

== Features ==
 
* Fully responsive and mobile ready.
* Strong Options Panel.   
* Easy Shortcode
* Easy Installation. 
* Translation ready 
* Easy to use and customize


== Installation ==

1. Go to Plugins > Add New.
1. Search for "WPxon Ajax Contact Form".
1. Click "Install Now".

OR

1. Download the zip file.
1. Upload the zip file via Plugins > Add New > Upload.

Activate the plugin. Look for "Ajax Contact Form" in the admin menu.

== Uses ==

There are 2 ways to publish Ajax Contact Form anywhere in the posts or pages with the following shortcodes:

= In posts or pages editor :- =
<code>[ajax_contact_form]</code>

= In php file:- =
<code><?php echo do_shortcode('[ajax_contact_form]'); ?></code>
